# Quizit

A quiz app for studying questions from books and topics. Track your progress, save favorites, and review questions you got wrong.

## What You Can Do

- **Sign up or log in** with your email and password
- **Browse books** – see available books and how many questions each has
- **Open topics** – each book has topics you can practice
- **Take topic quizzes** – answer questions one at a time, with explanations after each answer
- **Bookmark** – save your place so you can continue later
- **Favorite questions** – add or remove favorites to review them anytime
- **Review favorites** – practice all your favorite questions in one place
- **Review mistakes** – practice questions you got wrong; answering correctly removes them from the list
- **Track progress** – see how many questions you’ve answered per topic

## Getting Started

1. Create an account or log in
2. Choose a book from the list
3. Pick a topic to start a quiz
4. Answer questions and use the favorite and bookmark buttons as needed

## Quizzes

- **Topic quiz** – questions from a single topic. You can bookmark your position and favorite any question.
- **Favorites** – all favorited questions from the book. Add or remove favorites as you go.
- **Mistakes** – all questions you got wrong. Answer them correctly to remove them from this list.

## Navigation

- Use **Back** (top left) to return to the previous page
- Use **Sign out** (top right) to log out
- Use **Previous** and **Next** in the quiz to move between questions
